## Description
ft_transcendence est un projet du cursus 42 consistant à développer une application web complète autour d’un jeu Pong en ligne.

L’application permet aux utilisateurs de créer un compte, de jouer à Pong en temps réel contre d’autres joueurs et d’interagir avec différentes fonctionnalités sociales.

## Objectif
Ce projet permet d'apprendre à :
- développer une application web full-stack
- gérer l’authentification et les utilisateurs
- implémenter des communications en temps réel
- concevoir une architecture backend / frontend
- déployer et organiser une application web complète

## Technologies
- JavaScript / TypeScript
- Node.js
- Fastify
- SQLite
- HTML / CSS
- Docker

## Compilation
docker compose up --build

## Utilisation
Accéder à l’application depuis le navigateur :
